A collection of Python-based data analytics projects developed as part of my learning journey in data analysis, statistical modeling, and data visualization. This repository contains projects demonstrating data processing, exploratory data analysis (EDA), statistical analysis, and visualization using popular Python libraries.

---

## Projects

### 1. Demographic Data Analyzer

A data analysis project that explores demographic datasets to generate statistical insights and answer analytical questions using Python.

---

### 2. Mean, Variance and Standard Deviation

A statistical analysis project that computes descriptive statistics and demonstrates fundamental concepts in numerical data analysis.

---

### 3. Medical Data Visualizer

A visualization project that analyzes medical examination data using categorical plots and heatmaps to identify trends and relationships.

---

### 4. Page View Time Series Visualizer

A time series analysis project that visualizes website traffic data using line plots, bar charts, and box plots to identify trends and seasonality.

---

### 5. Sea Level Predictor

A regression-based project that analyzes historical sea level data and predicts future trends using linear regression.
